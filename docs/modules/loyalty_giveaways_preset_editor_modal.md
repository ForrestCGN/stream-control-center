# Loyalty Giveaways – Preset Editor Modal

Der Preset-Editor ist als wiederverwendbares Dashboard-Modal angelegt.

## UI-Konzept
- Kein Browser-Popup.
- Großes Modal im Dashboard.
- Ein Editor für Presets und Giveaway-Wheel-Kontext.
- Schnellanlage bleibt bestehen, aber der Modal-Editor ist der bevorzugte Weg.

## Kontexte
- `presets`: globales Preset erstellen/bearbeiten.
- `giveaways`: Preset aus dem Giveaway-Formular heraus erstellen.

## Aktuelle Einschränkung
Die Runtime nutzt noch nicht vollständig eigene Bound-Wheel-Felder für Spins. Das bleibt ein späterer Step.
