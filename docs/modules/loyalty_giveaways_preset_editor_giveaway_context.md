# Loyalty Giveaways – Preset-Editor im Giveaway-Kontext

Der Preset-Editor ist eine gemeinsame UI-Komponente für den Presets-Tab und den Giveaway-Editor.

## Presets-Tab
- Globale Presets erstellen und bearbeiten.
- Felder ergänzen und pflegen.

## Giveaway-Kontext
- Bei Wheel-Giveaways kann ein bestehendes Preset ausgewählt werden.
- Alternativ kann über das Modal ein neues Preset erstellt werden.
- Das neue Preset wird automatisch im Giveaway-Formular ausgewählt.
- Das Modal bleibt im Bearbeiten-Modus, damit Felder ergänzt werden können.

## Readiness
Ein Wheel-Giveaway ist erst bereit, wenn ein gültiges Preset bzw. Rad mit mindestens einem aktiven gültigen Feld vorhanden ist und alle weiteren Pflichtdaten erfüllt sind.
