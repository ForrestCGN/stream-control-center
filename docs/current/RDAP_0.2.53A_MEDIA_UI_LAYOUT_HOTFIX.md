# RDAP 0.2.53A - Media UI Layout Hotfix

## Ziel

Layoutfehler aus 0.2.53 korrigieren. Die Media-Seite soll wieder nutzbar sein, ohne Backend-/Sync-Logik zu veraendern.

## Aenderungen

- Sync-Status als kompakte Full-Width-Karte.
- Medienliste als Full-Width-Karte unterhalb des Sync-Status.
- Hauptkarten zeigen Typ, Anzeigename/Dateiname, Bereich und Groesse.
- Pfad/relativePath/publicPath bleiben im Info-Fenster.
- Filter, Suche, Sortierung, Paging bleiben aktiv.

## Nicht enthalten

- Keine DB-Migration.
- Keine DB-Writes.
- Keine API-Aenderung.
- Keine Upload/Edit/Delete-Funktion.
- Keine Agent-Actions.
