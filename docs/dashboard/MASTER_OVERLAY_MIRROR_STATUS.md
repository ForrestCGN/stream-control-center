# Dashboard — Master Overlay Mirror Status

## Anzeigen

```text
Master Overlay connected
Mirror mode active/inactive
Alerts mirrored count
Last mirrored alert
Last mirror ack
Last mirror error
Legacy production target status
```

## Wichtig

Mirror-Fehler sind Test-Fehler, keine Produktionsfehler.

Dashboard sollte klar unterscheiden:

```text
Production Alert Overlay
Master Mirror Overlay
```

## Buttons

Später möglich:

```text
Enable Alert Mirror
Disable Alert Mirror
Send Test Alert to Master only
Send Last Alert to Master again
Open Master Overlay URL
```
