# Current Chat Handoff – LWG-4N.5

Bestätigte UX-Richtung:

- Loyalty ist eine Zentrale, kein einzelner Glücksrad/Giveaway-Tab.
- Bereiche: Übersicht, Punkteverwaltung, Glücksrad, Giveaways, Raffles, Texte, Statistik, Config, Verlauf.
- Presets liegen UI-seitig unter Glücksrad.
- Giveaway-Glücksräder werden nur im jeweiligen Giveaway bearbeitet.
- Config bleibt zentral.
- Commands werden im zentralen Command-Editor bearbeitet; Loyalty darf später nur fehlende Standard-Commands prüfen/anlegen, nie überschreiben.
- Texte laufen zentral über die Multi-Textverwaltung/Helper.

LWG-4N.5 enthält nur Dashboard-UI-Foundation, keine Backend-Änderung.
