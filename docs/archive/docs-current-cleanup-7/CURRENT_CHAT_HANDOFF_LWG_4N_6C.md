# Handoff – LWG-4N.6c

Bestätigter Kontext:
- Loyalty UI ohne Games-Zwischenebene.
- Giveaways als Kachelübersicht.
- Erstellen/Bearbeiten per Modal.

Neu in 4N.6c:
- Wheel-Giveaway-Kacheln zeigen je nach Entwurfszustand `Glücksrad erstellen` oder `Glücksrad bearbeiten`.
- `Glücksrad erstellen` öffnet den bestehenden Preset-/Glücksrad-Editor im Giveaway-Kontext.
- Nach dem Speichern wird das neue Glücksrad automatisch dem Giveaway zugeordnet.
- Offene/laufende Giveaways zeigen keine Edit-Aktionen mehr.
