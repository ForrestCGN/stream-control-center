# Overlay Monitoring – STEP627B

STEP627B korrigiert ausschließlich das Design des Rahmen-Overlays.

Der Rahmen ist weiterhin ein CGN-Overlay mit EventBus-Anmeldung und Heartbeats über `overlay:frame_overlay`. Das Design wurde ruhiger gestaltet: keine Neon-Punkte, kein Runner-Effekt, stattdessen ein schlichter Neon-Rand im Stil des Deathcounter-Overlays mit etwas stärkerer unterer Kante.

Keine Backend-, Dashboard- oder OBS-Änderungen.
