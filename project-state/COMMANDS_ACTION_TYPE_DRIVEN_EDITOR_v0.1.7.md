# Commands Dashboard v0.1.7 — Action-Type Driven Editor

## Geändert

- `htdocs/dashboard/modules/commands.js`
- Projekt-/Moduldoku aktualisiert

## Kernpunkte

- Neuer Command startet mit: Song, Video, Text, Modul-Befehl, Benutzerdefiniert.
- Bestehender Command zeigt die gespeicherte Aktionsart als ersten Dropdown-Eintrag.
- Die Maske richtet sich nach der gespeicherten bzw. ausgewählten Aktionsart.
- Gespeicherte technische Werte werden nicht mehr als Sonderkasten gezeigt, sondern unter Erweitert.
- Katalogwerte werden erst mit „Ausgewählte Aktion übernehmen“ ins Formular geschrieben.
- Textausgabe bleibt vorbereitet für spätere zentrale Textverwaltung.
