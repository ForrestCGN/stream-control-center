# Channelpoints v0.7.3 — Text Reward + Redemption Polish

Dieser Stand rundet den lokalen Kanalpunkte-Flow ab.

## Enthalten

- Backend `MODULE_VERSION=0.7.3`, `MODULE_BUILD=text-reward-redemption-polish`.
- Dashboard `UI_VERSION=0.7.3`, `UI_BUILD=text-reward-redemption-polish`.
- Text-Rewards sind lokal ausführbar.
- Text-Einlösungen werden in `channelpoints_redemptions.result_json` gespeichert.
- `GET /api/channelpoints/text-execution-check?reward=<key>` ergänzt.
- Redemption-Verlauf zeigt eine kurze Ergebnisvorschau.

## Wichtig

Textgruppen sind vorbereitet, aber noch nicht an eine zentrale Textverwaltung angebunden. In diesem Stand wird kein Twitch-Chat automatisch beschrieben und Twitch wird nicht verändert.
