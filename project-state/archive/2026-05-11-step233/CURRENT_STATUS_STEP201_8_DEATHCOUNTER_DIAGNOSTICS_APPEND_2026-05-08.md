# STEP201.8 Append – Deathcounter V2 Diagnose-Endpunkte

Deathcounter V2 wurde auf den STEP201-Diagnose-Standard ergänzt.

Produktiver Prefix:

```text
/api/deathcounter/v2
```

Ergänzt:

```text
/status
/config
/settings
/routes
/integration-check
/reload
```

Bewusst nicht ergänzt:

```text
/api/deathcounter
/api/deathcounter-v2
/api/deathcounter_v2
/api/death-counter
```

Keine Counter-/Overlay-/Broadcast-/Reset-Logik wurde geändert.
