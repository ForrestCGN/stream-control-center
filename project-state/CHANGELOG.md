# Changelog

## HT3.8

- HypeTrain-Übersicht um Live-Status „Beim nächsten echten HypeTrain aktiv“ ergänzt.
- Start, Stufenaufstieg, Ende und Rekord werden kompakt als Status-Kacheln angezeigt.
- Start/Rekord können als „Sound bereit“ sichtbar sein.
- Level-Up/Ende bleiben als geplant/offen sichtbar, solange keine Sounds gesetzt sind.
- Keine Backend-/DB-Änderung.
